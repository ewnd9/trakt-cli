# trakt-cli

[![Build Status](https://travis-ci.org/ewnd9/trakt-cli.svg?branch=master)](https://travis-ci.org/ewnd9/trakt-cli)

Access trakt from CLI

## Install

```
$ npm install trakt-cli -g
```

## Usage

```
$ trakt --help

  Usage

  $ trakt <show-title>
  $ trakt --available
  $ trakt --available --json
```
